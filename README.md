# All-games-amb-ance-mod

This software adds a dynamic snow falling effect and a customizable transparent color overlay to your screen. It is designed to enhance visual ambiance and focus during gameplay.

## Features
- **Ambiance Overlay:** Adds a transparent color layer to your screen.
- **Dynamic Snow:** Real-time snow falling effect with adjustable speed.
- **Integrated Crosshair:** A static center-screen crosshair.
- **Customization:** Full control over transparency levels and effect speeds.

To ensure the application runs correctly, please follow these steps:

1. **Download:** Navigate to the **Releases** section on the right and download the `ambiance.mod.zip` file.
2. **Extract Everything:** Extract the entire contents of the ZIP file into a **single folder**.
   > **Note:** Do not move the `.exe` file out of this folder, as it needs the accompanying files to function.
3. **Run the App:** Open the folder and launch `KxAmbianceMod.exe`.
4. **Desktop Shortcut (Optional):** To access the app from your desktop, right-click `KxAmbianceMod.exe` and select **"Send to > Desktop (create shortcut)"**. This allows the app to stay with its core files while you launch it from your desktop.
5. **In-Game Settings:** Set your game (e.g., CS2) video settings to **"Fullscreen Borderless"** or **"Windowed"** mode for the overlay to appear.
